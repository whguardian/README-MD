* 无序列表  
    * 标记方法：列表名前**星号 + space**
* 有序列表
    * 标记方法：列名前**编号 + . + space**
	    1. 列名1
	    2. 列名2
* 多级列表
	* 每级列名前多加一个**tab**(**四个space**)
	    * 本列表是三级列表，列名前需要两个tab
* 块引用
	* 内容前添加 **>** 符号
	*  块的嵌套：每一层增加一个 **>**
	* 例：

>DotA2
>> 英雄
>>>力量英雄
>>>>末日使者 马格纳斯 哈斯卡 。。。。。
